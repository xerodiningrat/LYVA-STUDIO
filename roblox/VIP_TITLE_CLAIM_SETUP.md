# VIP Title Claim Setup

1. Taruh `MX_VIPTitleClaim.lua` ke folder `MX_Modules`.
2. Tempel isi `MX_Main_VIPClaim_PATCH.lua` ke script `MX_Main`.
3. Aktifkan `Allow HTTP Requests` di Roblox Studio.
4. Jalankan panel Discord dengan `/titile setup`.
5. User klik `Claim Title`, isi username Roblox dan custom title.
